Control Points Help

Control Points Usage:

  1. Create connections: Use connect mode to link nodes first
  2. Add control points: Click any connection to add orange control points (max 2 per connection)
  3. Add interpolation points: Alt+Click connection to add green interpolation points
  4. Drag to reshape: Drag any visible control point to bend and reshape curves
  5. Delete points: Double-click any control point or right-click → delete
  6. Toggle visibility: Shift+Click connection or use right-click menu
  7. Right-click connection: Opens full menu with all curve options
  8. Delete connection: Ctrl+Click (or Cmd+Click on Mac)
